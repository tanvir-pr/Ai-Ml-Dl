# Week 6: Tree-based models

**Month 2 · Classical machine learning**

## Purpose
Move from a **single interpretable tree** to **ensembles** that dominate tabular ML competitions and industry: random forests (bagging) and gradient boosting libraries.

## Topics covered (in order)

| # | Notebook | Focus |
|---|----------|--------|
| 1 | `01_decision_trees.ipynb` | CART, entropy vs Gini, splits, depth, overfitting |
| 2 | `02_random_forests.ipynb` | Bagging, random subspaces, variance reduction |
| 3 | `03_gradient_boosting.ipynb` | Boosting idea, XGBoost / LightGBM / CatBoost usage |
| 4 | `04_feature_importance.ipynb` | Importance scores, interpreting model behavior |

## Learning outcomes
- Compute and compare **entropy** and **Gini** impurity conceptually.
- Explain **bias–variance** for trees vs **Random Forest** vs **GBDT**.
- Tune **depth**, **learning rate**, and **n_estimators** without severe overfitting.
- Use **feature importance** cautiously (correlated features, causal limits).
