# Mini Project: Credit Card Fraud Detection

## Overview
Detect fraudulent transactions using tree-based models on imbalanced data.

## Dataset
Kaggle Credit Card Fraud dataset or similar.

## Steps
1. Handle class imbalance (SMOTE, class weights)
2. Train: Decision Tree, Random Forest, XGBoost
3. Focus on recall (catch frauds)
4. Tune thresholds
5. Compare models, analyze feature importance

## Deliverables
- Trained models
- Confusion matrix, precision, recall, F1
- Feature importance plot
- Recommendations
