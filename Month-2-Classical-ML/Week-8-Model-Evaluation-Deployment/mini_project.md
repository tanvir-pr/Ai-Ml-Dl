# Mini Project: End-to-End ML Pipeline

## Overview
Build complete pipeline from data to deployment.

## Steps
1. Load data, preprocess
2. Train model with cross-validation
3. Hyperparameter tuning
4. Save model (joblib)
5. Create Flask API for predictions
6. Test API with requests
7. Optional: Dockerize

## Deliverables
- Trained model file
- API code
- Test script
- README with usage
