# Mini Project: Customer Segmentation

## Overview
Segment customers using clustering for marketing strategy.

## Dataset
Mall Customer or similar (age, income, spending score).

## Steps
1. EDA and feature scaling
2. Determine k (elbow, silhouette)
3. K-Means clustering
4. Profile each segment
5. Business recommendations per segment
6. Optional: PCA for visualization

## Deliverables
- Clustering results
- Segment profiles
- Visualizations
- Actionable insights
