# Week 7: Unsupervised learning

**Month 2 · Classical machine learning**

## Purpose
Discover **structure without labels**: clusters, low-dimensional representations, and anomalies. Uses linear algebra from Week 1 (PCA / SVD) in application.

## Topics covered (in order)

| # | Notebook | Focus |
|---|----------|--------|
| 1 | `01_kmeans_clustering.ipynb` | K-means, centroids, elbow, k choice |
| 2 | `02_hierarchical_clustering.ipynb` | Linkage, dendrogram, agglomerative clustering |
| 3 | `03_pca_dimensionality_reduction.ipynb` | Variance explained, projection, visualization |
| 4 | `04_anomaly_detection.ipynb` | Isolation Forest and anomaly mindset |

## Learning outcomes
- Choose and justify **k** in K-means vs interpret a **dendrogram** cut.
- Interpret **PCA** loadings and **explained variance ratio**.
- Contrast **clustering** (similarity groups) vs **anomaly detection** (rare points).
- Name when **DBSCAN** / **GMM** (exercises) beat K-means.
