# Mini Project: Email Spam Detector

## Overview
Build a spam/ham classifier using text preprocessing and classification algorithms.

## Dataset
Use UCI Spam dataset or create from sample emails.

## Steps
1. Load and preprocess text (lowercase, tokenize, remove stopwords)
2. TF-IDF or Count vectorization
3. Train: Naive Bayes, Logistic Regression
4. Evaluate: accuracy, precision, recall, F1
5. Compare models, tune hyperparameters

## Deliverables
- Trained model
- Evaluation metrics
- Example predictions
- Brief report
