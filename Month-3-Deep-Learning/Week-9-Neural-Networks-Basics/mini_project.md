# Mini Project: MNIST Digit Classification

## Overview
Classify handwritten digits (0-9) using neural networks.

## Dataset
MNIST: 60k train, 10k test, 28x28 grayscale images.

## Steps
1. Load and preprocess (normalize)
2. Build MLP (Keras or PyTorch)
3. Train with validation
4. Evaluate on test set
5. Visualize misclassifications
6. Try improvements (dropout, more layers)

## Target
Accuracy > 98%

## Deliverables
- Trained model
- Training curves
- Evaluation metrics
- Brief report
