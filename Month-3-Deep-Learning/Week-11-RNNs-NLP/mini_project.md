# Mini Project: Movie Review Sentiment Analyzer

## Overview
Classify movie reviews as positive/negative using deep learning.

## Dataset
IMDB dataset (or Rotten Tomatoes)

## Steps
1. Load and preprocess
2. Tokenize and pad
3. Build model (LSTM or BERT)
4. Train with validation
5. Evaluate on test set
6. Optional: Deploy as API

## Target
Accuracy > 85%

## Deliverables
- Trained model
- Evaluation metrics
- Sample predictions
- Brief report
