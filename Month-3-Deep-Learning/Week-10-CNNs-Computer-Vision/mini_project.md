# Mini Project: Custom Image Classifier

## Overview
Build image classifier for custom categories using transfer learning.

## Dataset
Dog breeds, flowers, or your own images (100+ per class).

## Steps
1. Organize data (train/val folders)
2. Use ImageDataGenerator with augmentation
3. Load pre-trained model (MobileNet, ResNet)
4. Add custom classification head
5. Fine-tune or train top layers
6. Evaluate and visualize predictions

## Deliverables
- Trained model
- Accuracy metrics
- Sample predictions
- Brief report
