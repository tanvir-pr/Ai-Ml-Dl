# Capstone Project Options

## Choose ONE to complete your 3-month journey:

### Option 1: End-to-End Image Classification
- Custom dataset (1000+ images)
- CNN with transfer learning
- Deploy as web app (Streamlit/Gradio)
- Documentation and README

### Option 2: NLP Chatbot with Sentiment
- Build chatbot using transformers
- Integrate sentiment analysis
- Simple conversation flow
- API or web interface

### Option 3: Recommendation System
- User-item interactions
- Matrix factorization or neural collaborative filtering
- Recommend top-k items
- Evaluate with metrics (precision@k, NDCG)

### Option 4: Time Series Forecasting
- LSTM/Transformer for forecasting
- Stock prices, sales, or weather
- Train, validate, predict
- Visualization of predictions

## Requirements for Capstone
- Real dataset
- Complete pipeline (data -> model -> evaluation)
- Documented code
- README with setup and usage
- GitHub repository
- Brief report (findings, challenges, learnings)

## Evaluation
- Technical implementation (40%)
- Documentation (20%)
- Creativity/novelty (20%)
- Presentation (20%)
