# ML & DL — 3-month learning path

Hands-on notebooks from **Python & math** through **classical ML** to **deep learning** (12 weeks).

## Start here

1. **Read [LEARNING_PATH.md](LEARNING_PATH.md)** — setup, folder layout, and the full week-by-week list (this is the only navigation guide you need).  
2. **Install:** `pip install -r requirements.txt` (use a virtual environment).  
3. **Open Jupyter:** `jupyter lab` and go to **`Month-1-Foundations/Week-1-Python-Math-Basics/`** — start with `01_python_refresher.ipynb`.
