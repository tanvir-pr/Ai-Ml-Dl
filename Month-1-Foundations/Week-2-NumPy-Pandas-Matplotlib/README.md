## 💡 Key Concepts

### NumPy
```python
# Array operations
arr = np.array([1, 2, 3, 4, 5])
arr * 2  # Vectorized operation

# Broadcasting
matrix + vector  # Automatic shape alignment

# Indexing
arr[arr > 3]  # Boolean indexing
```

### Pandas
```python
# DataFrame operations
df = pd.read_csv('data.csv')
df.groupby('category').mean()

# Data cleaning
df.fillna(df.mean())
df.drop_duplicates()

# Merging
pd.merge(df1, df2, on='key')
```

### Visualization
```python
# Matplotlib
plt.plot(x, y)
plt.xlabel('X')
plt.ylabel('Y')
plt.show()

# Seaborn
sns.scatterplot(data=df, x='x', y='y', hue='category')
```
