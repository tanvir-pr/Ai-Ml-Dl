# Mini Project: A/B Test Analysis

## Overview
Analyze an A/B test to determine if a new website design improves conversion rates.

## Objectives
1. Set up null and alternative hypotheses
2. Perform statistical tests
3. Calculate confidence intervals
4. Make data-driven recommendations

## Dataset
Create or use simulated data:
- Control group: 1000 users, 50 conversions (5%)
- Treatment group: 1000 users, 65 conversions (6.5%)

## Steps
1. **Formulate hypotheses**: H0: p_control = p_treatment
2. **Chi-square test** or **two-proportion z-test**
3. **Calculate 95% CI** for difference in conversion rates
4. **Interpret results**: Is the improvement significant?
5. **Recommendation**: Should we launch the new design?

## Deliverables
- Jupyter notebook with analysis
- Statistical test results
- Visualization of conversion rates
- Written recommendation report

## Evaluation
- Correct hypothesis formulation
- Appropriate test selection
- Proper interpretation of p-value
- Clear recommendation
