   - Where will you use these concepts?

   - 1-2 hours daily is better than 10 hours once

5. **Ask Questions**
   - Use Stack Overflow and Reddit
   - Join study groups
   - Don't struggle alone for too long

## Common Pitfalls to Avoid

1. **Rushing Through Math**
   - Take time to understand concepts deeply
   - Math is the foundation - don't skip it

2. **Not Implementing from Scratch**
   - Using libraries is easy
   - Understanding comes from implementation

3. **Passive Learning**
   - Watching videos without coding
   - Reading without practicing

4. **Perfectionism**
   - Don't get stuck on one problem too long
   - Move forward, come back later

5. **Isolated Learning**
   - Connect new concepts to previous knowledge
   - See the big picture

## Quick Reference Cards

### Python One-Liners
```python
# List comprehension
squares = [x**2 for x in range(10)]

# Dictionary comprehension
square_dict = {x: x**2 for x in range(10)}

# Filter with lambda
evens = list(filter(lambda x: x % 2 == 0, range(10)))

# Map with lambda
doubled = list(map(lambda x: x * 2, range(10)))

# Zip for parallel iteration
for x, y in zip(list1, list2):
    print(x, y)
```

### NumPy Quick Reference
```python
# Create arrays
a = np.array([1, 2, 3])
A = np.array([[1, 2], [3, 4]])

# Array operations
A.T                    # Transpose
A @ B                  # Matrix multiplication
A * B                  # Element-wise multiplication
np.linalg.inv(A)      # Inverse
np.linalg.det(A)      # Determinant
np.linalg.eig(A)      # Eigenvalues/vectors
```

### Calculus Quick Reference
```python
# Numerical derivative
def derivative(f, x, h=1e-5):
    return (f(x + h) - f(x)) / h

# Gradient descent
def gradient_descent(f, grad_f, x0, lr=0.01, n_iter=100):
    x = x0
    for _ in range(n_iter):
        x = x - lr * grad_f(x)
    return x
```
