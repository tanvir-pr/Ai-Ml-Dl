| `mini_project.md` | Weekly project |
| `resources.md` | Links / further reading |
| `README.md` | Week intro (where present) |
| `capstone_project.md` | Week 12 capstone options only |
| `next_steps.md` | Week 12 follow-up only |
| `*_executed.ipynb` | Optional saved run of a notebook (duplicate) |
